## What is it?

Stelgent is an AI-powered platform that lets users generate code through prompts inside a VSCode-like IDE. It supports multi-turn conversations, context-aware file edits, and real-time project management.

## Key Features

### AI Code Generation
- Multi-turn contextual conversations
- Intelligent file updates without duplication
- Gemini 2.0 Flash powered generation

### VSCode-like IDE
- Monaco Editor
- File tree explorer
- Syntax highlighting
- Split panels and real-time editing

### Project Management
- Multiple projects
- CRUD operations on files
- Separate conversation history per project

### Security
- JWT Authentication
- Bcrypt password hashing
- MongoDB isolation
- CORS protection
